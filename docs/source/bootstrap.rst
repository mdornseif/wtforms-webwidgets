Bootstrap
=========


.. automodule:: wtforms_webwidgets.bootstrap
.. attribute:: default_widgets
    
    A dictionary defining the default widget type for each kind
    of WTForms Field. 

    Very useful when using FieldRenderer to automatically render
    your fields without declaring ``widget=MyWidget()``.
